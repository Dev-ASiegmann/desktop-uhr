# desktop-uhr
Kleine Uhr für den Desktop. Die Größe ist anpassbar. Die Uhr ist verschiebbar. 09.06.2026
Das Programm sollte ab Windows 10 aufwärts laufen. Vielleicht sogar ab Windows 7. 
Bei den ersten Installationen kommt vermutlich eine Windows Warnung. Diese kann man mit weiter bestätigen.
Der Virenscanner wird vermutlich beim ersten Aufruf anspringen und die Datei scannen. Diesen einfach durchlaufen lassen.
